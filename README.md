# DevOps AWS

avance de proyecto
## Tecnologías utilizadas

- GitHub
- Linux
- Bash
- Python + Boto3
- AWS CloudFormation
- Docker
- AWS CodePipeline
- AWS CodeBuild
- AWS CloudWatch

## Estructura

- bash/: scripts automatizados
- python/: automatización con boto3
- docker/: contenedores
- cloudformation/: infraestructura como código
- docs/: documentación final
- screenshots/: evidencias

## CI/CD Pipeline

This project is prepared for CI/CD integration using:

- GitHub as source repository
- AWS CodeBuild for build automation
- AWS CodePipeline for deployment workflow
